  • About Vanguard Force

    Vanguard Force is a small project I started when i discovered Pavlov VR.
    I realized that there are lot of old Call of Duty maps but no servers that were running them frequently.

    Because of that I started my first Pavlov VR server on Vultr hosting provider and soon after that i decided to expand, now i run 3 servers, on bare metal server.

    Server Spec

    Processor: AMD Ryzen 7 4800U, 8 Cores / 16 Threads
    Memory: DDR4 Dual channel (3200mhz), 32GB
    Network: 1 Gbit/s
